QR Code Generator Using Python
📌 Overview

QR Code Generator is a Python-based application that allows users to generate QR codes from text, URLs, email addresses, phone numbers, and other information. The generated QR codes are saved as image files and can be scanned using any QR code scanner or smartphone camera.

This project demonstrates the use of Python libraries for QR code generation and image processing while providing a simple and efficient solution for sharing information digitally.

🚀 Features
Generate QR codes from text and URLs
Save QR codes as PNG images
Fast and lightweight application
Easy-to-use command-line interface
Custom file name support
High-quality QR code generation
Beginner-friendly Python project


🛠 Technologies Used
Python 3.x
qrcode Library
